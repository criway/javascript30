exercise number 3: "Playing with CSS variables and JS"
we can update variables in CSS with javascript.
	CSS:
		Creacion variables: se crean cn el ":root{}" y con el prefijo "--", ejemplo:
				:root{
					--base: #ffc600;
					--spacing: 10px;
					--blur: 10px;
				}
				Para despues usarlas en el codigo, debemos usarlas con "variable(--<variableName>)" tambien.
		"input type range": input de tipo barra
		"input type color": input de tipo color, al seleccionarlo se abre una ventana para seleccionar un color.
		"label for": relaciona una etiqueta o el label con un elemento de tipo input
